1. Write statements for each of the following
a. Store ten random integers within 0 to 1000 to a text file name integer.txt.

```java

    }

```

b. Read from the text file generated in a. Display all the integer and the largest
integer.

```java
try{
    Scanner scanner = new Scanner(new FileInputStream("a.txt"));
    int max = Integer.MIN_VALUE;
    while(scanner.hasNextInt()){
        int number = scanner.nextInt();
        System.out.print(number + " ");
        if(number > max){
            max = number;
        }
    }
    System.out.println("\nLargest Integer: " + max);

}catch(FileNotFoundException e){
    e.printStackTrace();
}
```

c. Store ten random integers within 0 to 1000 to a binary file name integer.dat.

```java
try{
    ObjectOutputStream outputStream = new ObjectOutputStream(new FileOutputStream("integer.dat"));
    for(int i=0;i<10;i++){
        outputStream.writeInt((int)(Math.random()*1001));
    }
    outputStream.close();

}catch(IOException e){
    e.printStackTrace();
}
```

d. Read from the binary file generated in a c. Display the all the integer and the
average.

```java
int average=0,no=0;
try{
    ObjectInputStream inputStream = new ObjectInputStream(new FileInputStream("c.dat"));
    while(true){
        int number = inputStream.readInt();
        System.out.print(number + " ");
        average+=number;
        no+=1;
    }

}catch(EOFException e){
    System.out.printf("\nAverage: %.2f",(double)average/no);
}catch(IOException e){
    e.printStackTrace();
}catch(FileNotFoundException e){
    e.printStackTrace();
}
```

3. Write a program that convert a sentence into binary number (ASCII code 8 bit) and
store it in a text file named data.txt. Then, read from the text file and display the
sentence.
```java
try{
    ObjectOutputStream outputStream = new ObjectOutputStream(new FileOutputStream("data.txt"));
    outputStream.writeUTF("this is a sentence.");
    outputStream.close();

    ObjectInputStream inputStream = new ObjectInputStream(new FileInputStream("data.txt"));
    String sentence = inputStream.readUTF();
    System.out.print(sentence);
}catch(IOException e){
    e.printStackTrace();
}
```
